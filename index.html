Robotic Automated Arm and Control System

This project features a fully automated robotic arm built and programmed to perform precise, repeatable movements using microcontroller‑based control logic. The system integrates multiple servos, sensor feedback, and custom motion routines to execute tasks with accuracy and stability.

Arduino Ultrasonic Radar

This project uses an Arduino Uno, HC‑SR04 ultrasonic sensor, SG90 servo motor, and a breadboard to create a simple scanning radar system. The servo sweeps the ultrasonic sensor across a set angle range, while the Arduino measures distance using timed echo pulses. Each reading is processed in the loop and sent over serial to visualize detected objects in real time. The system demonstrates basic sensor interfacing, servo control, timing functions, and serial communication in Arduino.
